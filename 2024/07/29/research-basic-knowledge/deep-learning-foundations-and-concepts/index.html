Likelihood function

给定一个有限集合 (a Finite set of observations) x=(x1,…,xN)\mathsf{x} = (x_1, \dots, x_N)x=(x1​,…,xN​), 估计一个分布的问题被称为密度估计. 密度估计是一个不适定问题, 因为存在无穷多个概率分布可以采样出这个有限集合

适定问题, 满足以下所有条件的问题

  1. 解必须存在

  2. 解必须唯一

  3. 解能根据初始条件连续变化，不会发生跳变，即解必须稳定

从同一分布中独立抽取的数据点称为独立同分布, 通常记为 i.i.d\mathrm{i.i.d}i.i.d 或 IID\mathrm{IID}IID

独立同分布的一组数据, 多个独立事件的联合概率是由每个事件的边缘概率的成绩给出, 所以我们可以将这组数据的概率密度相乘, 得到似然函数

p(x∣μ,σ2)=∏n=1NN(xn∣μ,σ2)p(\mathsf{x}|\mu, \sigma^2) = \prod^N_{n=1}\mathcal{N}(x_n|\mu, \sigma^2) p(x∣μ,σ2)=n=1∏N​N(xn​∣μ,σ2)

最大似然估计: 使用观测数据集 (an observed data set), 寻找使似然函数最大化的参数值, 确定概率分布中的参数.

Conditional entropy

考虑联合概率分布 p(x,y)p(\mathbf{x}, \mathbf{y})p(x,y), 如果 x\mathbf{x}x 已知, 那么指定相应 y\mathbf{y}y 值所需的额外信息为 ln⁡p(y∣x)\ln p(\mathbf{y|\mathbf{x}})lnp(y∣x). 其平均值, 也叫做条件熵的定义如下

H[y∣x]=−∫∫p(y,x)ln⁡p(y∣x)dydx\mathrm{H}[\mathbf{y}|\mathbf{x}] = -\int \int p(\mathbf{y}, \mathbf{x})\ln p(\mathbf{y}|\mathbf{x})\mathrm{d}\mathbf{y}\mathrm{d}\mathbf{x} H[y∣x]=−∫∫p(y,x)lnp(y∣x)dydx

此外可以得到,

H[x,y]=H[y∣x]+H[x]−∫∫p(x,y)ln⁡p(x,y)dxdy=−∫∫p(y,x)ln⁡p(y∣x)dydx−∫p(x)ln⁡p(x)dx\mathrm{H}[\mathbf{x}, \mathbf{y}] = \mathrm{H}[\mathbf{y}|\mathbf{x}] + \mathrm{H}[\mathbf{x}]\\ -\int\int p(\mathbf{x}, \mathbf{y})\ln p(\mathbf{x}, \mathbf{y})\mathrm{d}\mathbf{x}\mathrm{d}\mathbf{y} = -\int \int p(\mathbf{y}, \mathbf{x})\ln p(\mathbf{y}|\mathbf{x})\mathrm{d}\mathbf{y}\mathrm{d}\mathbf{x} -\int p(\mathbf{x})\ln p(\mathbf{x})\mathrm{d}\mathbf{x} H[x,y]=H[y∣x]+H[x]−∫∫p(x,y)lnp(x,y)dxdy=−∫∫p(y,x)lnp(y∣x)dydx−∫p(x)lnp(x)dx

Mutual information

两个变量独立时, 他们的联合概率分布等于他们边缘分布的乘积 p(x,y)=p(x)p(y)p(\mathbf{x}, \mathbf{y}) = p(\mathbf{x})p(\mathbf{y})p(x,y)=p(x)p(y). 如果两个变量不独立, 可以通过联合分布与边缘分布的概率密度函数乘积间的 KL 散度来判断他们是否接近独立. xxx 和 yyy 间的互信息定义为

I[x,y]≡KL(p(x,y)∥p(x)p(y))=−∫∫p(x,y)ln⁡(p(x)p(y)p(x,y))dxdy\begin{aligned} \mathrm{I}[\mathbf{x}, \mathbf{y}] \equiv \mathrm{KL}(p(\mathbf{x}, \mathbf{y})\| p(\mathbf{x})p(\mathbf{y}) ) &= -\int\int p(\mathbf{x}, \mathbf{y})\ln \left(\frac{p(\mathbf{x})p(\mathbf{y})}{p(\mathbf{x}, \mathbf{y})} \right)\mathrm{d}\mathbf{x}\mathrm{d}\mathbf{y} \end{aligned} I[x,y]≡KL(p(x,y)∥p(x)p(y))​=−∫∫p(x,y)ln(p(x,y)p(x)p(y)​)dxdy​

互信息和条件熵有关

I[x,y]=H[x]−H[x∣y]=H[y]−H[y∣x]\mathrm{I}[\mathbf{x}, \mathbf{y}] = \mathrm{H}[\mathbf{x}] - \mathrm{H}[\mathbf{x}|\mathbf{y}] = \mathrm{H}[\mathbf{y}] - \mathrm{H}[\mathbf{y}|\mathbf{x}] I[x,y]=H[x]−H[x∣y]=H[y]−H[y∣x]

互信息的意义是在得知 y\mathbf{y}y 的值后, x\mathbf{x}x 的不确定性减少. 可以从贝叶斯公式的角度理解, p(x)p(\mathbf{x})p(x) 是先验分布, p(x∣y)p(\mathbf{x}|\mathbf{y})p(x∣y) 是后验分布, 计信系是由于后验的存在导致的先验的不确定性的减少

Bayesian Probabilities

使用概率来表示不确定性是很好的选择. 如果使用数值, 那么常识性的公理都最终将等价于概率的加法和惩罚规则

Model parameters

我们用 D\mathcal{D}D 表示训练数据集, w\mathbf{w}w 是使似然函数 p(D∣w)p(\mathcal{D}|\mathbf{w})p(D∣w) 最大化的值. 似然函数的负对数被称为误差函数, 因为负对数是一个单调递减函数, 最大化似然函数等同于最小化误差, 这样得到 wML\mathbf{w}_{\mathrm{ML}}wML​.

假设参数 w\mathbf{w}w 满足先验概率 p(w)p(\mathbf{w})p(w), 观测数据 D\mathcal{D}D 的影响通过 p(D∣w)p(\mathcal{D}|\mathbf{w})p(D∣w) 进行表达. 贝叶斯定理现在是

p(w∣D)=p(D∣w)p(w)p(D)p(\mathbf{w}|\mathcal{D}) = \frac{p(\mathcal{D}|\mathbf{w})p(\mathbf{w})}{p(\mathcal{D})} p(w∣D)=p(D)p(D∣w)p(w)​

贝叶斯定理能让我们通过后验概率评估 w\mathbf{w}w 的不确定性. 当 p(D∣w)p(\mathcal{D}|\mathbf{w})p(D∣w) 看做关于 w\mathbf{w}w 的函数时, 这也可以看做似然函数.

Bayesian machine learning

贝叶斯的视角让我们看到正则化的作用, 并推导出正则化项的具体形式. 但贝叶斯定理寻找 w\mathbf{w}w 的单一解, 没有考虑到 w\mathbf{w}w 值的不确定性. 假设我们有训练数据集 D\mathcal{D}D, 我们想在给定 x\mathrm{x}x 的情况下预测某个目标变量 ttt. 显然有

p(t∣x,D)=∫p(t∣x,w)p(w∣D)dwp(t|x, \mathcal{D}) = \int p(t|x, \mathbf{w})p(\mathbf{w}|\mathcal{D})\mathrm{d}\mathbf{w} p(t∣x,D)=∫p(t∣x,w)p(w∣D)dw

这里 p(w∣D)p(\mathbf{w}|\mathcal{D})p(w∣D) 可以看做是加权参数. 这种方法与贝叶斯方法的关键差异是对参数空间的积分.

完全贝叶斯式的机器学习方法更具有可解释性. 例如多项式回归背景下的过拟合问题, 就是最大似然法的使用中出现的问题. 如果我们用贝叶斯方法对参数进行边缘化处理, 这一问题就不会出现. 比如回归问题中不同阶数的多项式回归, 最大似然法倾向于选择对训练数据拟合更好的模型, 但是这导致模型复杂度更好, 从而会产生过拟合. 完全贝叶斯的处理方法则是对所有可能的模型进行平均, 每个模型的贡献按其后验概率加权, 这使得复杂度适中的模型更容易被选中, 这相当于优雅的惩罚了复杂性.

但贝叶斯的方法也存在计算量巨大的问题. 所以可以考虑对神经网络使用最大似然技术, 并使用正则化技术来避免过拟合
